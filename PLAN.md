# TODO LIST
- Simple TODO LIST webapp.
- No need to create account.

## Frontend
- Use Vite.
- Use ReactJS + TailwindCSS + Typescript.
- Responsive.

## Backend
- Use NodeJS + ExpressJS + Typescript
- API Versioning.
- Routes, controllers, services and repositories.
- Must have `/api/v1/health`.
- Use `express-validator` for validation.

## Testing
- Unit Testing: Jest.
