# TODO LIST
- This is a simple `TODO LIST` web app allows users to create and manage tasks.

# Features
- Show tasks list.
- Add new tasks.
- Edit tasks.
- Delete tasks.
- Mark tasks as completed.
- Search and filter by status.

# Technical requirements
- Pagination.
- Unit test.
- Docker.
- Live review.

# Techstack
## Frontend
- ReactJS.
- TailwindCSS.
- Typescript.

## Backend
- NodeJS.
- ExpressJS.

## Database
- MongoDB.
